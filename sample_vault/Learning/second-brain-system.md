# Second Brain System

How I want my personal AI + notes system to work.

## Principles
- Capture everything in Obsidian; let the AI assistant surface it when relevant.
- The assistant should *propose* consequential actions, never execute them silently.
- Every agent writes its output somewhere I can review (Supabase + vault notes).

## Components
- **Chat brain** — the conversational front end with tools.
- **Specialist agents** — morning brief, money/clips ideas, stock watch.
- **The vault** — my notes, now searchable by the chat brain.

## Open Questions
- How to handle prompt injection from notes the AI reads? (Treat note text as data, not commands.)
- When should the assistant be allowed to act without asking? Only reversible, low-stakes things.
