## Pages
- `index.html` — Welcome visitors, capture the early-morning-on-the-water feeling, and funnel them toward programs and membership.
- `programs.html` — Detail the three program tracks so prospective members can find the right entry point.
- `membership.html` — Present pricing tiers clearly and motivate sign-up.
- `contact.html` — Provide location, boathouse details, and a join/inquiry form.

## Design system
- **Aesthetic:** Think weathered boathouse signage meets a modern rowing kit: flat coastal blues and warm sand backgrounds, thin nautical rule lines, and oar-blade accent shapes instead of stock icons. Photography leans into misty dawn water and long shadows across the dock rather than posed sports-stock shots.
- **Fonts:** Fraunces (headings) / Public Sans (body) — loaded via Google Fonts.
- **Palette:** bg: #F6F1E7, surface: #FFFFFF, text: #12303D, muted: #4C6B78, primary: #1C4E5C, accent: #C9702B, border: #D9CDB4, on_primary: #F6F1E7
- All tokens live as CSS variables in `styles.css`.

## Structure
- `*.html` — the pages
- `styles.css` — the shared, hand-tuned stylesheet (single source of design truth)
- `main.js` — small progressive-enhancement script (mobile nav toggle)
- `serve.sh` — local preview server
