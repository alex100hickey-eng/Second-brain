## Pages
- `index.html` — Single-page site covering the product story, features, proof, pricing and signup for the Inkling note-taking app

## Design system
- **Aesthetic:** Feels like a well-kept notebook reimagined for screens: warm cream paper background, a single confident ink-blue accent, and a slightly imperfect hand-set heading font paired with a crisp, legible body face. Sections breathe like margins on a page rather than dense app-marketing blocks, with thin rule-lines standing in for notebook divisions instead of heavy card borders.
- **Fonts:** Fraunces (headings) / Inter (body) — loaded via Google Fonts.
- **Palette:** bg: #FBF6EC, surface: #F3ECDC, text: #2A2622, muted: #6B6255, primary: #1E3A5F, accent: #D9622B, border: #DDD3BC, on_primary: #FBF6EC
- All tokens live as CSS variables in `styles.css`.

## Structure
- `*.html` — the pages
- `styles.css` — the shared, hand-tuned stylesheet (single source of design truth)
- `effects.css` — guaranteed motion/cinematic layer, loaded after styles.css (kept separate so it always works)
- `main.js` — progressive-enhancement script (mobile nav, scroll-reveal animations, FAQ accordion, sticky-nav shadow)
- `serve.sh` — local preview server
