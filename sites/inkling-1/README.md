Notes that think with you.

## Preview locally
```bash
bash serve.sh          # then open http://localhost:8080
# or: bash serve.sh 9001   to use a different port
```

## Pages
- `index.html` — Single-page marketing site that introduces Inkling, shows how it works, builds trust, and converts students to download or sign up.

## Design system
- **Aesthetic:** Feels like a well-kept notebook re-imagined for screens: warm cream paper tones, a single confident inkwell-blue accent, and hairline rules instead of heavy boxes. Editorial, restrained, and calm — closer to a stationery brand than a SaaS dashboard, with plenty of breathing room between ideas.
- **Fonts:** Fraunces (headings) / Inter (body) — loaded via Google Fonts.
- **Palette:** bg: #FBF7EF, surface: #F3ECDD, text: #241F1A, muted: #6B6255, primary: #274B8C, accent: #C1622A, border: #DED2B8, on_primary: #FBF7EF
- All tokens live as CSS variables in `styles.css`.

## Structure
- `*.html` — the pages
- `styles.css` — the shared, hand-tuned stylesheet (single source of design truth)
- `effects.css` — guaranteed motion/cinematic layer, loaded after styles.css (kept separate so it always works)
- `main.js` — progressive-enhancement script (mobile nav, scroll-reveal animations, FAQ accordion, sticky-nav shadow)
- `serve.sh` — local preview server
