## Pages
- `index.html` — Give a warm, appetizing first impression of the restaurant and drive visitors to the menu and visit info.
- `menu.html` — Present the full menu clearly and appetizingly, organized by category with realistic items, descriptions, and prices.
- `visit.html` — Make it effortless for someone to decide when to come, how to get there, and what to expect once inside.

## Design system
- **Aesthetic:** Picture the glow of an open hearth at dusk: deep charcoal walls, warm ember-orange light spilling across a cream tablecloth. Headlines are set in a characterful, slightly rough-edged serif reminiscent of hand-painted restaurant signage, paired with a clean humanist sans for body copy so menu items stay legible. Photo placeholders are framed like small windows into the wood-fired oven — warm-toned, softly shadowed, never sterile.
- **Fonts:** Fraunces (headings) / Karla (body) — loaded via Google Fonts.
- **Palette:** bg: #221A17, surface: #2E2420, text: #F3E9DC, muted: #C9B8A8, primary: #E2652B, accent: #F0A15C, border: #4A3B33, on_primary: #221A17
- All tokens live as CSS variables in `styles.css`.

## Structure
- `*.html` — the pages
- `styles.css` — the shared, hand-tuned stylesheet (single source of design truth)
- `main.js` — small progressive-enhancement script (mobile nav toggle)
- `serve.sh` — local preview server
